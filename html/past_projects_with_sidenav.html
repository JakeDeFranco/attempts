2021 - COVID-19 Vaccine Tracker Web Application

Myself, and a team of 4 others build a real-time vaccine tracker web application during a company wide hackathon. This application would track the number of people that have been vaccinated, fully and partially, in the countries, regions and cities of each of Point's offices. This information would be used to calculate a best estimate of when the Point's employees could be going back into the office.

The front end was built using React.js, HTML, CSS and Yarn? while the back end consisted of Python, and a MySQL database. Real time vaccine information was gathered by HTTP API calls.

Tech Stack: Python, Flask, HTML & Docker
